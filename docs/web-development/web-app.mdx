# 6. Webアプリとは

Webアプリケーション（Webアプリ）は、ウェブブラウザ上で動作するソフトウェアアプリケーションのことを指します。
みなさんがスマホやPCでインストールするアプリ(ネイティブアプリ)とは異なり、Webアプリはインターネットを通じてアクセスされ、ブラウザ上で実行されます。
ブラウザ上で開けるアプリはすべてWebアプリと考えて差し支えありません。授業中にPCで見るTwitterとかめちゃくちゃWebアプリやね。

## 6.1 Webアプリの仕組み
Webアプリは、主にフロントエンドとバックエンドの2つの部分で構成されています。
- **フロントエンド**: ユーザーが直接操作する部分です。HTML、CSS、JavaScriptなどの技術を使用して構築され、ユーザーインターフェースを提供します。
- **バックエンド**: サーバー側で動作する部分です。データベースとのやり取り、ユーザー認証などを担当します。

ユーザーがWebアプリにアクセスすると、ブラウザはフロントエンドのコードをサーバーから取得し、表示します。
ユーザーの操作に応じて、フロントエンドはバックエンドと通信し、必要なデータを取得または送信します。

## 6.2 実際にWebアプリを作ってみる
ここでは、簡単なWebアプリを作成してみましょう。
以下のコードは、HTML、CSS、JavaScriptを使用して作成されたシンプルなWebアプリの例です。

```html
<!DOCTYPE html>
<html>
  <head>
    <title>シンプルWebアプリ</title>
    <style>
      body {
        font-family: Arial, sans-serif;
        margin: 20px;
      }
      #output {
        margin-top: 20px;
        font-weight: bold;
      }
    </style>
  </head>
  <body>
    <h1>シンプルWebアプリ</h1>
    <input type="text" id="inputText" placeholder="テキストを入力してください" />
    <button onclick="showOutput()">表示</button>
    <div id="output"></div>
    <script>
      function showOutput() {
        const input = document.getElementById('inputText').value;
        document.getElementById('output').innerText = `入力されたテキスト: ${input}`;
      }
    </script>
  </body>
</html>
```

このコードを例のように保存してブラウザで開くと、テキスト入力フィールドとボタンが表示されます。
テキストを入力してボタンをクリックすると、入力されたテキストが画面に表示されます。

:::warning 注意
まあこれくらいのコードならAIが0.2秒で書いてくれます。とんでもない。
:::

このWebアプリでは、ユーザの操作に応じてJavaScriptが動作し、動的に内容が更新される仕組みを体験できます。
このようなユーザの操作の介在によって動的に変化することを **インタラクティブ** と呼びます。

## 6.3 SPAとMPA
現代のWebアプリには主に2つのやり方があります。
- **MPA (Multi-Page Application)**: 従来のWebアプリの形態で、各ページが個別のHTMLファイルとして存在します。ユーザーがリンクをクリックすると、新しいページがサーバーから読み込まれます。
- **SPA (Single-Page Application)**: 1つのHTMLファイルで構成され、必要に応じてJavaScriptで内容を動的に更新します。ページ遷移がなく、ユーザー体験が向上します。

難しいと思いますが、例えばTwitterやFacebookのようなアプリはSPAの例です。twitterでツイートをクリックしてもページ全体がリロードされず、スムーズに内容が切り替わりますね。あれはSPAの特徴です。
これに対して、WikipediaのようなサイトはMPAの例です。リンクをクリックするとページ全体がリロードされ、新しいページが表示されます。

ここまで聞くとSPAの方が良さそうに思えますが、実際には用途や要件に応じて使い分けられます。`ashitabo.net`もSPAとMPAを組み合わせて使っています。なんとなく。

## 6.4 まとめ
- Webアプリはウェブブラウザ上で動作するソフトウェアアプリケーション
- フロントエンドとバックエンドの2つの部分で構成される
- HTML、CSS、JavaScriptを使用してユーザーインターフェースを構築
- ユーザーの操作に応じて動的に内容が更新される仕組みを持つ
