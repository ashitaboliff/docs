# 環境変数のセットアップ
[第3章](./make.mdx)ではあしたぼホームページのコードをクローンして、VSCodeで開くところまで説明しました。
ここでは、あしたぼホームページのコードを動かすために必要な環境変数のセットアップについて説明します。

## 1. 環境変数とは
環境変数とは、プログラムが動作する環境に関する情報を格納するための変数のことです。  
あしたぼホームページでは、APIキーやデータベースの接続情報など、漏洩したらクソまずい情報を環境変数として管理しています。

## 2. 環境変数のセットアップ
あしたぼホームページのコードを動かすためには、必要な環境変数をセットアップする必要があります。  
フロントエンドは、`.env.example`ファイルをコピーして`.env`ファイルを作成してください。
```bash
# フロントエンドの環境変数設定
cd ~/ashitabo/app/frontend
cp .env.example .env
```

バックエンドは環境変数を暗号化してgit管理しています。まずは以下のコマンドを実行してあなたの秘密鍵と公開鍵を生成してください。
```bash
age-keygen -o ~/.config/sops/age/keys.txt
```

これを実行すると
```bash
Public key: age1...
```
のような出力がされるので、`age1...`の部分をコピーしておいてください。ここで出てきたのは他人に知られても良い公開鍵です。

そうしたら、`app/backend`リポジトリでブランチを切りましょう！練習したように`dev`ブランチから`feat/○○-env`みたいな感じでブランチを切ってください。

次に、`app/backend/.sops.yaml`というファイルをVSCodeで開きましょう。すると
```yaml
creation_rules:
  - path_regex: \.env\.dev\.enc$
    age: 
      - age1... # watabegg's Key

  - path_regex: \.env.(prd|stg)\.enc$
    age: 
      - age1... # SRE's Key
```
みたいな感じで公開鍵が書いてあると思います。ここに先ほどコピーした公開鍵を追記してください。
```yaml
creation_rules:
  - path_regex: \.env\.dev\.enc$
    age: 
      - age1... # watabegg's Key
      - age1... # あなたの公開鍵

  - path_regex: \.env.(prd|stg)\.enc$
    age: 
      - age1... # SRE's Key
```

`#`から始まるコメントの部分は`# {GitHubのID}'s Key`みたいな感じで書いておいてください。  
そうしたらこの変更をコミット/プッシュして、GitHub上で`dev`ブランチへのプルリクエストを作成してください！
以前と同様にレビュアーにwatabeggを追加してDiscordでレビューを依頼してください。
ここで鍵の登録が行われると、やっと環境変数の暗号化ファイルを復号できるようになります。

:::tip
gitでのこの操作フローは[前章](./git-test.mdx)に詳しく載っています！忘れてたらこっちを見ながら慎重に進めましょう
:::

## 3. 環境変数の復号と立ち上げ
さきほどのプルリクエストがマージされたら、`app/backend`リポジトリのルートにある`.env.dev.enc`というファイルを復号して`.env.local`というファイルを作成します。
```bash
cd ~/ashitabo/app/backend
sops decrypt ./secrets/.env.dev.enc > .env.local
```

これで環境変数のセットアップが完了しました！ではフロントエンドとバックエンドを立ち上げてみましょう！

:::tip
この説明ではターミナル1画面での操作を記述していますが、VSCodeではターミナルを複数開いて管理することが出来ます。
僕はFE, BE, types, docsそれぞれでターミナルを開いていることが多いです。便利だよ。
:::

それぞれのディレクトリに移動し、以下のコマンドを実行してください。
```bash
# バックエンドの起動
cd ~/ashitabo/app/backend
pnpm install
pnpm dev
```
バックエンドが正常に起動したら、別のターミナルを開いて以下のコマンドを実行します。
```bash
# フロントエンドの起動
cd ~/ashitabo/app/frontend
pnpm install
pnpm dev
```
ブラウザで`http://localhost:3000`にアクセスし、あしたぼホームページが表示されることを確認してください。
これで開発環境のセットアップと動作確認は完了です。次に、VSCodeでのコードの見方について説明します。
