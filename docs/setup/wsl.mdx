# WSLでの環境構築

WindowsでWeb開発を行う場合、WSL(Windows Subsystem for Linux)を利用することが推奨されます。 WSLを使用することで、Linux環境をWindows上で簡単に利用でき、多くの開発ツールやライブラリがLinux向けに最適化されているため、開発がスムーズに行えます。

:::tip
MacやLinuxを使っている人はWSLのインストールは不要です。
:::

## 1. WSLのインストール

まず、WSLをインストールします。まあ調べれば色々出ますが躓かないように以下の手順に従ってください。
<StepList>
1. **PowerShellを管理者権限で開く**  
   スタートメニューから「PowerShell」を検索し、右クリックして「管理者として実行」を選択します。
2. **WSLのインストールコマンドを実行**  
   以下のコマンドをPowerShellに入力して実行します。
   ```powershell
   wsl --install
   ```
   :::info
   「ダウンロードしています：Ubuntu」とか出てきたらいい感じです。
   :::
3. **初回起動とユーザー設定**  
   インストールすると自動でUbuntuが起動するはずなので初回セットアップを行います。ユーザー名とパスワードを設定してください。
   :::warning
   Ubuntuが起動しない場合は「PowerShell」(管理者権限で開く必要はありません)から以下のコマンドを実行して起動してください。
   ```powershell
   wsl
   ```
   :::

   多分こんな感じのプロンプトが出るはずです。入力してください。
   ```powershell
   Provisioning the new WSL instance Ubuntu
   This might take a while...
   Create a default UNIX user account: <ここにユーザー名を入力>
   New password: <ここにパスワードを入力>
   Retype new password: <もう一度パスワードを入力>
   ```
   まあなんか違っても大丈夫です。とりあえずユーザー名とパスワードを設定してください。
   :::tip
   ユーザー名とパスワードはなんでもいいですが、忘れないようにしてください。
   僕はユーザー名は`watabegg`にしててパスワードは`******`にしてます。そこまでセキュリティは求められてないと思います！多分
   :::
4. **WSLのバージョン確認**
   初期設定が終わったら、なんかPowerShellに色が付きましたね？ それはWSLが起動している証拠です。次に
   ```
   exit
   ```
   と入力してWSLを終了します。再びWindowsのPowerShellに戻ったのでWSLのバージョンを確認します。以下のコマンドを実行してください。
   ```bash
   wsl -l -v
   ```
   もしWSLのバージョンが1になっていたら、以下のコマンドでバージョン2にアップグレードしてください。
   ```bash
   wsl --set-version Ubuntu 2
   ```
   :::tip
   再度WSLを起動するには、PowerShellで
   ```bash
   wsl
   ```
   と入力してください。そんだけでOK！
   :::
</StepList>

WSLのインストールは以上です。この時点で詰まったらすぐに教えて下さい。

## 2. 開発環境のセットアップ

WSLがインストールできたら、次に開発環境をセットアップします。以下の手順に従ってください。
curlやgitなどの基本ツールはUbuntuを入れれば付いてくるのでそれでいいです。
<StepList>
1. **WSLを起動**  
   PowerShellで以下のコマンドを実行してWSLを起動します。
   ```bash
   wsl
   ```
2. **Node.jsのインストール**  
   Node.jsというJavaScriptをPC上で実行するための環境をインストールします。以下のコマンドを実行してください。
   ```bash
   curl https://get.volta.sh | bash
   source ~/.bashrc
   volta install node@lts
   volta install pnpm
   ```
   :::tip
   Node.jsはなんかいろんなバージョンがあってそれを管理するためにVoltaというツールを使っています。Voltaを使うとプロジェクトごとにNode.jsのバージョンを変えられたりして便利です。  
   他にもいろいろ`nvm`とかあるけどこれが一番楽そう。
   :::
3. **動作確認**  
   Node.jsとpnpmが正しくインストールされたか確認するために、以下のコマンドを実行してバージョン情報が表示されることを確認してください。
   ```bash
   node -v
   pnpm -v
   ```
   ちゃんとバージョンが表示されたら成功です！
</StepList>
