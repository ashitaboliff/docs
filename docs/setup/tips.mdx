# 開発のヒント
開発環境のセットアップが完了したら、いよいよコードを書いていく段階になります。
ここでは、あしたぼホームページの開発を進めていく上で知っておくと便利なヒントをいくつか紹介します。

## 1. GitHub Copilotの利用
GitHub Copilotは、GitHubが提供するAI Agentで、コードの自動生成や補完を行うツールです。
学生はGitHub Proに無料で登録できて、そこそこのトークンをCodexやClaude CodeなどのAI Agentに使えるので、ぜひ利用してみてください。

[学生無料のCopilot を使うために GitHub Education に登録 - Qiita](https://qiita.com/melonsode/items/3602ea6441ca82e43c5a)を参考にしてください。

## 2. AI Agent用ファイルの活用
あしたぼホームページのリポジトリには、AI Agentに対してプロジェクトの構造やコードの説明を提供するためのファイルがいくつか用意されています。  
これらのファイル、AIに対してもわかりやすくなっていますが、人間に対してもわかりやすいように書いてあったりします。`AGENTS.md`や`README.md`などを読んで、プロジェクトの全体像やコードの構造を把握してみてください。

## 3. コードの品質管理
あしたぼホームページのコードは、コードの品質を保つために**biome**というツールが導入されています。  
このツールは「コードのインデントを適切にする」「規約違反のコードに対してエラーを吐く」などを行ってくれます。
